## 项目简介

**DB-personal-web** 是一个单文件纯原生个人品牌网站，融合 **粒子物理引擎**、**SVG 图腾动画**、**CSS 3D 轨道系统** 等前端实验性表达，以沉浸式叙事展示 AI 工具深度实践、内容创作运营与全栈开发三条能力主线。

> 配色体系：`#0E0E0E` / `#C5A880` / `#F5F4F0` · 字体：Syne · Bodoni Moda · Plus Jakarta Sans

   ▼ 同频共振             →  联系方式 + 拖拽徽章

## 本地运行

```bash
git clone https://github.com/H-Canopy/DB-personal-web.git
cd DB-personal-web

open DB-web.html        # macOS
start DB-web.html       # Windows
# 或
npx serve .
```

> ⚡ 单文件 · 零依赖 · 零构建 — 浏览器直接打开即可
