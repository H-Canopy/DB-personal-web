## ✦ 项目简介

**DB-personal-web** 是我的个人品牌网站，承载了我在 AI Agent 探索、内容创作运营、全栈开发三个方向上的实践与思考。网站采用 **黑金配色体系**，通过 Canvas 粒子系统、SVG 动画、CSS 3D 变换等前端技术，构建了一个沉浸式的个人品牌叙事空间。

## ✦ 视觉亮点

| 模块 | 技术实现 | 说明 |
|------|---------|------|
| **鎏金圣纹开屏** | SVG + CSS Animation | 丁达尔光效、金粒飘散、幕布拉开式入场动画 |
| **粒子 Dark Hero** | Canvas 2D | 鼠标驱散粒子交互、二进制数字飘落纹理、随机格言浮层 |
| **奇点轨道图** | SVG + CSS Keyframes | 技术标签沿椭圆轨道旋转、奇点脉冲光晕、多层深度叠加 |
| **双行 Marquee** | CSS Animation | 正反向无限滚动、关键词强调 |
| **神话太阳图腾** | SVG + CSS Animation | 四层火焰摇曳动画、日轮光晕 |
| **拖拽徽章** | JS Drag API | SVG 文字路径环绕、可拖拽互动 |
| **鼠标跟随太阳** | JS + SVG | 光标实时跟随、双层火焰反向旋转 |
| **滚动渐入动画** | Intersection Observer | fade-up 序列化入场 |

## ✦ 本地运行

```bash
# 克隆仓库
git clone https://github.com/H-Canopy/DB-personal-web.git

# 进入目录
cd DB-personal-web

# 直接用浏览器打开
open DB-web.html
# 或使用任意静态服务器
npx serve .
```

纯静态 HTML 文件，零依赖，无需构建工具，浏览器直接打开即可预览。

## ✦ 设计理念

- **黑金主色调**：黑色代表技术深度与专注，香槟金 (#C5A880) 注入温度与质感
- **东西方美学融合**：鎏金圣纹符号、神话太阳图腾 与 Syne/Bodoni Moda 西文字体并置
- **叙事性交互**：从开屏仪式感到逐段内容展开，构建完整的浏览节奏
- **性能优先**：纯原生实现，无框架开销，Canvas 与 CSS 动画合理分层
